# AoDuo户外活动管理系统

## 环境

+ nodejs
  > 百度搜索 对应平台版本安装
+ npm
  > nodejs 安装好后自带
+ yarn
  > 在命令行使用命令 :  npm i -g yarn

## 使用步骤(命令行中完成)

```shell
  # 进入项目根目录   
  
  cd aodu
  
  # 安装依赖
  
  yarn install
  
  
  
  # 等待依赖安装完成
  
  # 运行
  
  yarn run serve
  
  
  # 按照提示打开浏览器
  
  

```

# 打包和部署

```shell
  # 打包
  yarn build
  
  # 更目录的dist即打包好后的目录
  # 使用nginx部署就好
```
